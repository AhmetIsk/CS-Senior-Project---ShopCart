About This Page

This page is created for our senior project ShopCart. ShopCart is a mobile app that creates a shopping list for you and bring you the cheapest products! Create your group, scan the depleted product to the cam and do not forget anything in the list anymore! Keep your spending data, take some advice and stay healthy!

Project Reports

Project Specifications Report

11/10/2021

This report includes introduction, description, ethical issues, several constraints, funtional and non-functional requirements for the ShopCart app.

Project Analysis Report

15/11/2021

This report examines current and proposed systems, functional and non-functional systems and contains diagrams of ShopCart such as use case, sequence, activity and examplifies possible scenarios.

High Level Design Report

24/12/2021

This report contains high level design of ShopCart. It proposes a software architecture that contains system decomposition, hardware software mapping, access control and security, etc.

Final Report

This report is the final documentation of the ShopCart project. It is the summation of the product and presentation with the combination of previous reports.
